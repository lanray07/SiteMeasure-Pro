# SiteMeasure Pro AI Privacy Policy

Effective date: May 19, 2026

SiteMeasure Pro AI helps contractors create local project measurements, estimates, and proposal drafts on iPhone and iPad.

## Data Handling

The current app stores project details, client names, property addresses, notes, site photos, measurements, material estimates, labor estimates, subscription state, and generated proposal file references locally on the user's device using Apple platform storage.

Mock AI mode is enabled by default. In this mode, site photos and project details are not sent to the developer or to an AI provider.

If a future production AI backend is enabled, uploaded photos, project notes, and measurement context may be sent to a secure backend endpoint to generate estimates. API keys should not be stored in the iOS app.

## Purchases

Subscriptions and digital purchases are processed by Apple through StoreKit. The developer does not receive or store payment card details.

## Tracking

The app does not use third-party advertising tracking and does not sell personal data.

## User Control

Users can delete projects and generated local data from within the app when deletion features are available, or remove the app from their device to delete locally stored app data. Users should verify all measurements and estimates before using them for client quotes.
